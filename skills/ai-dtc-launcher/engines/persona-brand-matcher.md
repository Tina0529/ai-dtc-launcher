# 人群-品牌匹配法 (Persona-Brand Matcher)

## 用途

从目标品类的用户评论数据中提取人群画像标签，匹配审美偏好和参考品牌风格，最终输出可直接交付给设计师的"品牌视觉 DNA 简报"。让 AI 完成从用户洞察到品牌视觉方向的全链路推导。

---

## 执行步骤

### Step 1: 从用户评论提取画像标签

使用以下 prompt 对采集到的用户评论/笔记数据进行画像标签提取：

```
你是一个消费者洞察分析师。请分析以下用户评论/笔记数据，提取目标消费人群的画像标签。

**提取维度（每个维度给出 1-3 个最突出的标签）：**

1. **年龄段**: 推测用户的大致年龄范围（如 18-24/25-30/30-40/40+）
2. **性别倾向**: 男性为主/女性为主/中性
3. **消费能力**: 学生党/小白领/中产/高净值
4. **生活方式**: 如独居/合租/家庭/宝妈/健身党/养生/数码控/宠物人群...
5. **审美偏好**: 从内容中推断用户的视觉审美倾向（见下方审美标签库）
6. **购买动机**: 自用/送礼/尝鲜/刚需替换/跟风种草
7. **信息渠道**: 用户主要通过什么渠道了解产品（小红书/抖音/朋友推荐/直播/搜索）
8. **价格敏感度**: 极度敏感/中等/不敏感（根据评论中对价格的讨论判断）

**输出格式：**

### 核心人群画像

| 维度 | 主标签 | 副标签 | 判断依据（引用原文） |
|------|--------|--------|-------------------|
| 年龄段 | {tag} | {tag} | "原文引用..." |
| 性别倾向 | {tag} | — | "原文引用..." |
| 消费能力 | {tag} | {tag} | "原文引用..." |
| 生活方式 | {tag} | {tag} | "原文引用..." |
| 审美偏好 | {tag} | {tag} | "原文引用..." |
| 购买动机 | {tag} | — | "原文引用..." |
| 信息渠道 | {tag} | {tag} | "原文引用..." |
| 价格敏感度 | {tag} | — | "原文引用..." |

### 一句话人群描述

用一句话概括目标人群："{年龄}{性别}{生活方式}，追求{审美偏好}风格，通过{渠道}种草，{消费能力}水平。"

---

以下是用户评论/笔记数据：

{在此粘贴数据}
```

### Step 2: 匹配审美标签

根据 Step 1 提取的审美偏好标签，从以下审美标签库中匹配最贴合的 1-2 个标签：

#### 中国市场审美标签库

| 标签 | 关键词线索 | 视觉特征 | 代表品牌参考 |

| **氛围感** | 有氛围、好出片、仪式感、精致 | 暖色调，柔光，细节丰富 | 观夏、闻献、野兽派 |
| **松弛感** | 随性、舒服、自然、不刻意 | 大地色系，天然材质，不规则排版 | 内外NEIWAI、素然 |
| **国潮** | 中国风、传统、文化、国货 | 红/金/墨色，书法元素，东方纹样 | 花西子、李宁中国行、毛戈平 |
| **极简** | 简洁、没有多余、实用、克制 | 黑白灰，无衬线字体，几何形状 | MUJI、网易严选 |

### Step 3: AI 匹配参考品牌

使用以下 prompt 让 AI 基于人群画像和审美标签匹配最合适的参考品牌：

```
你是一个品牌策略顾问。基于以下消费人群画像和审美偏好，推荐 3-5 个参考品牌，并说明每个品牌值得借鉴的具体方面。

**目标人群画像**:
{粘贴 Step 1 的一句话人群描述}

**审美偏好标签**:
{粘贴 Step 2 匹配的审美标签}

**目标品类**:
{品类名称}

**要求**:

1. 推荐的参考品牌不必限于同品类，重点是审美调性和人群匹配度
2. 每个品牌需说明：
   - 品牌名称和所属品类
   - 为什么审美风格匹配目标人群
   - 具体可借鉴的视觉元素（配色/字体/包装/拍摄风格/社交媒体视觉）
   - 不适合借鉴的地方（避免完全照搬）
3. 最后给出一个综合的视觉方向建议

**输出格式**:

### 参考品牌推荐

| 序号 | 品牌名 | 品类 | 匹配度 | 可借鉴方面 | 注意事项 |
|------|--------|------|--------|-----------|---------|
| 1 | {brand} | {category} | ★★★★★ | {aspect} | {caveat} |
| ... | ... | ... | ... | ... | ... |

### 各品牌详细分析

#### 品牌 1: {brand_name}
- **匹配理由**: ...
- **借鉴要素**: ...
- **避免照搬**: ...

...

### 综合视觉方向建议

{一段话描述建议的整体视觉方向}
```

### Step 4: 视觉 DNA 提取

基于前 3 步的结果，沿以下 5 个维度提炼品牌的"视觉 DNA"，形成可直接交付设计师的简报：

#### 维度 1: 配色方案

```
主色: {色值/色名} — 用于 Logo、主视觉
辅色: {色值/色名} — 用于背景、辅助元素
点缀色: {色值/色名} — 用于 CTA、亮点强调
禁忌色: {色值/色名} — 品牌绝不使用的颜色
配色情绪: {一个词概括，如"温暖克制"/"冷酷科技"/"自然活力"}
```

#### 维度 2: 字体方向

```
标题字体: {类型} — 如圆体/黑体/宋体/手写体/无衬线/衬线
正文字体: {类型}
字体情绪: {如"亲和力"/"专业感"/"文艺气质"}
中文字体参考: {具体字体名，如思源黑体/站酷快乐体}
英文字体参考: {具体字体名，如Futura/Garamond/Montserrat}
```

#### 维度 3: 调性词

```
品牌调性词（3-5 个）: {如"温暖 · 专业 · 自然 · 可信赖"}
反调性词（绝不是）: {如"廉价 · 浮夸 · 冰冷"}
一句话调性: "{品牌名}是_____的，不是_____的。"
```

#### 维度 4: 场景

```
核心使用场景（1-2 个）: {如"清晨护肤仪式"/"办公室下午茶"}
内容拍摄场景: {如"自然光窗边/白色桌面/绿植背景"}
禁忌场景: {如"过度PS的完美场景/嘈杂凌乱的背景"}
```

#### 维度 5: 材质感

```
包装材质倾向: {如"磨砂/哑光/天然纸质/透明玻璃"}
视觉材质感: {如"纸张纹理/水彩渲染/金属光泽"}
触感联想: {如"温润/轻盈/扎实"}
```

---

## 注意事项

1. Step 1 的评论数据应与 brand-vacuum-detector 使用相同的数据集，避免重复采集。
2. 审美标签库覆盖了主流风格，但如果用户评论明确指向某个未列出的风格（如"赛博朋克"），可以自行定义新标签。
3. 参考品牌推荐不限于同品类——跨品类的审美借鉴往往更有差异化效果。
4. 视觉 DNA 简报是品牌建设的起点文件，后续所有设计工作（Logo/包装/详情页/社交内容）都应以此为基准。
5. 不同市场使用对应的审美标签库，不要将中国审美标签用于日本市场，反之亦然。
