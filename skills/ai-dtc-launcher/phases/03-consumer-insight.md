# Phase 3: 消费者透视 (Consumer Insight)

## 阶段目标

深入理解目标消费者的真实需求，发现品牌真空和竞品弱点。通过大量用户声音的采集与分析，找到差异化切入点和品牌建设机会。

---

## 输入

| 参数 | 来源 | 说明 |
|------|------|------|
| 选定产品 | Phase 2 输出 | 1-3 个候选产品名称 |
| 品类信息 | Phase 1 输出 | 品类名称/竞品/价格带 |
| 头部玩家清单 | Phase 1b 输出 | Top 10 品牌/卖家 |
| 目标市场 | meta.json | china / japan / global |

---

## 执行步骤

### Step 1: 用户声音采集

#### 1.1 内容平台评论采集

**抖音/小红书/B站评论（Top 200 条）**：

Semi-auto 模式（抖音/小红书）：
```
提示用户：
"请在以下平台找到与 '{product_name}' 相关的 Top 5 高互动视频/笔记，
复制每个内容下的前 40 条评论（共约 200 条）粘贴给我。

优先选择：
- 测评类内容（用户会分享使用体验）
- 好物推荐类（用户会问具体使用问题）
- 吐槽/避坑类（用户会暴露真实痛点）

复制格式：
[视频/笔记标题]
评论1: ...
评论2: ...
..."
```

Auto 模式（B站）：
- 使用 WebSearch 搜索 B站 相关视频
- 提取评论和弹幕中与产品相关的用户声音

#### 1.2 电商平台评论采集

**淘宝/拼多多（好评 100 条 + 差评 100 条）**：

Semi-auto 模式：
```
提示用户：
"请在淘宝/拼多多搜索 '{product_name}'，找到销量 Top 3 的商品。
每个商品复制：
- 好评（4-5星）前 30-40 条
- 差评（1-2星）前 30-40 条

关键：差评比好评更有价值！请重点采集差评。

或者截图给我，我来识别提取。"
```

#### 1.3 日本市场评论采集（如适用）

使用 `amazon-japan-shopping` 获取 Amazon JP 产品评论：

```
搜索参数：
- 关键词: {product_name_jp}
- 重点采集: 1-2 星差评 + 4-5 星好评
```

WebSearch 搜索楽天评论：
```
搜索查询：
- "楽天 {product_name_jp} レビュー 口コミ"
- "楽天 {product_name_jp} 低評価"
```

### Step 2: AI 痛点聚类

对采集到的所有用户评论进行 AI 聚类分析。

**内置 Prompt**：

```
请将以下用户评论聚类为 5-8 个核心需求/痛点主题。

要求：
1. 每个主题用一个简短标签命名（如："尺寸不合适"、"材质触感差"、"性价比高"）
2. 每个主题标注倾向：正面需求 / 负面痛点 / 中性期待
3. 统计每个主题的评论数量和占比
4. 提取每个主题下 3-5 条最有代表性的原始评论
5. 识别被多次提及但现有产品未满足的需求（"未被满足的需求"）
6. 特别关注用户自发对比竞品的评论（品牌心智信号）

用户评论数据：
{reviews_data}
```

```markdown

| ... | ... | ... | ... |

### 负面痛点（用户不满的）

| 主题 | 提及次数 | 占比 | 代表性评论 |
|------|---------|------|-----------|
| {theme_1} | {N} | {pct}% | "{quote_1}" / "{quote_2}" / "{quote_3}" |
| ... | ... | ... | ... |

### 未被满足的需求

| 需求 | 提及频率 | 现有产品覆盖度 | 机会等级 |
|------|---------|--------------|---------|
| {need_1} | {frequency} | {低/中/高} | {高/中/低} |
| ... | ... | ... | ... |
```

### Step 3: 品牌真空检测

**操作**：加载 `engines/brand-vacuum-detector.md` 执行品牌真空检测。

> 如果 `engines/brand-vacuum-detector.md` 尚未创建，使用以下简化流程：

#### 品牌提及率分析

从 Step 1 采集的评论中统计品牌提及情况：

```
品牌提及率 = 提及任何品牌名称的评论数 / 总评论数

判定：
- 品牌提及率 < 5% → 品牌真空，用户无品牌偏好，极佳的品牌建设机会
- 品牌提及率 5-15% → 品牌弱势，有少数品牌但心智不牢固
- 品牌提及率 > 15% → 品牌密集，需要强差异化策略
```

| ... | ... | ... | ... | ... |

### Step 4: 竞品薄弱点扫描

从 5 个维度全面扫描竞品的弱点和可攻击面。

#### 4a 产品层：电商差评聚类

数据来源：淘宝/拼多多 1-2 星差评

```
AI 分析 Prompt：
"请将以下差评聚类为产品缺陷主题，按严重程度排序。
每个主题标注：
- 缺陷描述
- 影响的用户场景
- 是否可通过产品改进解决

差评数据：{negative_reviews}"
```

**输出：产品缺陷清单**

| 排名 | 缺陷主题 | 提及次数 | 严重程度 | 可改进性 | 改进方向 |
|------|---------|---------|---------|---------|---------|
| 1 | {defect} | {count} | {高/中/低} | {容易/可行/困难} | {suggestion} |
| ... | ... | ... | ... | ... | ... |

#### 4b 体验层：避坑笔记分析

使用 WebSearch 搜索小红书避坑内容：

```
搜索查询：
- "小红书 {product_name} 踩雷"
- "小红书 {product_name} 避坑"

```

提取营销话术反面教材：
- 虚假宣传的典型话术
- 买家秀 vs 卖家秀的差异
- 用户被"种草"后失望的原因

#### 4c 服务层：运营标准底线

搜索查询：

#### 4e 退出信号：二手转卖分析

**闲鱼分析**（中国市场）：

使用 WebSearch 搜索：
```
搜索查询：
- "闲鱼 {product_name} 转让"
- "闲鱼 {product_name} 全新"
```

**メルカリ分析**（日本市场，如适用）：

使用 WebSearch 搜索：
```
搜索查询：
- "メルカリ {product_name_jp} 出品"
- "メルカリ {product_name_jp} 新品未開封"
```

**退出信号报告**：

| 指标 | 数据 | 判定 |
|------|------|------|
| 转卖原因聚类 | {reasons} | {正常/预警} |
| 全新未拆占比 | {pct}% | {高(>30%)=冲动消费品类⚠️ / 低(<10%)=正常} |
| 二手保值率 | {rate}% | {>60%高保值✅ / 30-60%正常 / <30%溢价虚高⚠️} |
| 转卖量趋势 | {↑/→/↓} | {↑=退出增加⚠️ / →=正常 / ↓=留存好✅} |

**二手保值率计算**：
```
二手保值率 = 闲鱼/メルカリ均价 ÷ 原价(电商平台零售价)

判定：
- > 60%: 高保值 ✅ — 产品价值被市场认可
- 30-60%: 正常 — 合理折旧
- < 30%: 溢价虚高 ⚠️ — 原价可能包含过多品牌溢价或营销泡沫
```

## 失败路径

### 情况 1: 品牌提及率 > 15%（品牌密集）

```
不直接 No-Go，转差异化策略：
↓
分析品牌心智地图中的薄弱环节：
- 有无品牌形象老化？
- 有无价格带空白？
- 有无细分人群未被服务？
↓
找到差异化方向 → 标记为"差异化切入"模式，继续 Phase 4
找不到差异化方向 → 返回 Phase 2，该产品竞争壁垒过高
```

### 情况 2: 用户评论采集不足

```
可能原因：产品太小众 / 用户配合度低
↓
行动：
- 扩大搜索到上级品类的评论
- 使用竞品的评论作为代理数据
- 降低数据量要求，但标注"数据量有限，结论置信度中等"
```

### 情况 3: 退出信号预警（全新未拆占比 > 30%）

```
提示用户：
"⚠️ 该产品在闲鱼/メルカリ上全新未拆封的转卖比例较高（{pct}%），
说明消费者冲动购买后后悔的比例不低。

这意味着：
1. 可能是'种草效果强但实际体验差'的品类
2. 也可能是'礼品属性强'的品类（收到不需要的礼物转卖）

建议在 Phase 5 特别注意：
- 强化产品实际体验，缩小期望差
- 控制退货率预留到 15%+
- 详情页如实展示产品，避免过度美化

是否继续？"
```

## 注意事项

1. 用户评论是最有价值的数据源，宁可花时间多采集，不要跳过这一步
2. 差评比好评更重要——差评揭示了真实痛点和改进机会
3. 品牌真空检测的结果直接影响 Phase 4 的品牌策略：真空越大，品牌建设空间越大
4. 退出信号分析是独特的视角，大多数竞品分析都忽略了这一维度
5. 日本市场的品质期待通常高于中国市场，需要额外关注包装、细节和说明书的品质
6. 所有分析结论都需要有数据支撑，避免纯 AI 臆测
