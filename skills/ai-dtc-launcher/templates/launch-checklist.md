# 上线检查清单 (Launch Checklist)

## 用途

Phase 5 产品落地阶段的验收文档。在产品正式上线销售前，逐项确认所有准备工作已完成。所有项目必须打勾后方可进入 Phase 6 增长引擎阶段。

---

|------|------|

| **产品名称** | {product_name} |
| **目标上线日期** | {launch_date} |
| **首选销售渠道** | {primary_channel} |
| **检查日期** | {check_date} |

---

## 一、产品就绪

- [ ] **样品确认** — 收到供应商样品，确认材质/颜色/尺寸/做工符合要求
- [ ] **批量生产完成** — 首单 {quantity} 件已生产完毕
- [ ] **质检通过** — 逐件/抽检完成，不良率 < {threshold}%
- [ ] **产品实拍** — 拍摄实物照片，与效果图对比无重大偏差
- [ ] **包装确认** — 内外包装、吊牌、说明书、感谢卡齐全
- [ ] **条码/编码** — SKU 编码已分配，商品条码（如需）已申请

---

## 二、供应商确认

- [ ] **合同/协议签署** — 采购合同或订单确认函已签署
- [ ] **首批发货时间** — 确认发货日期：{ship_date}
- [ ] **退换货协议** — 明确退换货条件、责任划分、处理流程
- [ ] **质量标准约定** — 不良品处理方式已书面确认
- [ ] **补货响应时间** — 确认补货周期：{restock_days} 天
- [ ] **付款方式确认** — 付款节点和方式已明确（定金/尾款/账期）

## 三、店铺搭建

### 微信小商店（首选渠道）

- [ ] **小商店开通** — 微信小商店已注册并通过审核
- [ ] **商品上架** — 产品已创建并上架，信息完整
- [ ] **支付设置** — 微信支付已开通并绑定结算账户
- [ ] **运费模板** — 运费模板已设置（包邮/按重量/按地区）
- [ ] **售后规则** — 退换货政策、售后服务说明已配置
- [ ] **视频号关联** — 小商店已关联视频号，可挂链接

## 四、素材上架

- [ ] **白底主图** — 至少 1 张纯白底产品正面图（800×800 以上）

- [ ] **细节图** — 至少 2 张产品细节/材质特写
- [ ] **尺寸/规格图** — 1 张标注尺寸的示意图（如适用）

### 详情页

- [ ] **产品详情页** — 图文详情页已设计并上传
- [ ] **卖点提炼** — 从 Phase 3 痛点映射的 3-5 个核心卖点已体现

### SKU 信息

- [ ] **SKU 变体** — 颜色/尺码/规格等变体已创建完整
- [ ] **库存数量** — 各 SKU 库存已录入

### 上线判定

- **100% 完成** → 可以上线，进入 Phase 6
- **≥ 90% 完成** → 记录缺失项，评估是否可带缺上线
- **< 90% 完成** → 不建议上线，完成剩余项后重新检查

## 使用说明

1. 本清单由 Phase 5 产品落地阶段生成，作为进入 Phase 6 的门禁
2. 逐项打勾确认，不适用的项目标记 `N/A` 并说明原因
3. 所有必选项（无 N/A 标记）必须完成后方可进入 Phase 6
4. 建议将此清单打印或导出，团队/个人逐项确认签字
