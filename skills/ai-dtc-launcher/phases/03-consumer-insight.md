# Phase 3: 消费者透视 (Consumer Insight)

## 阶段目标

深入理解目标消费者的真实需求，发现品牌真空和竞品弱点。通过大量用户声音的采集与分析，找到差异化切入点和品牌建设机会。

---

## 输入

| 参数 | 来源 | 说明 |
|------|------|------|
| 选定产品 | Phase 2 输出 | 1-3 个候选产品名称 |
| 品类信息 | Phase 1 输出 | 品类名称/竞品/价格带 |
| 头部玩家清单 | Phase 1b 输出 | Top 10 品牌/卖家 |
| 目标市场 | meta.json | china / japan / global |

---

## 执行步骤

### Step 1: 用户声音采集

#### 数据采集分级方案

根据用户时间和需求，提供三个级别的数据采集方案：

| 级别 | 各平台采集量 | 合计评论数 | 预计耗时 | 结论置信度 |
|------|------------|-----------|---------|-----------|
| 快速版（最小可用） | 各平台 Top 30 条 | 约 50-80 条 | 约 30 分钟 | MEDIUM |
| 标准版（推荐） | 各平台 Top 50 条 | 约 100-150 条 | 约 1 小时 | HIGH |
| 深度版（完整） | 各平台 Top 100 条 | 约 200-300 条 | 约 2-3 小时 | VERY HIGH |

```
提示用户：
"评论数据是消费者透视的核心燃料。请选择数据采集级别：

1. 快速版（30 分钟）— 每平台 Top 30 条，适合快速验证想法
2. 标准版（1 小时，推荐）— 每平台 Top 50 条，结论可靠度高
3. 深度版（2-3 小时）— 每平台 Top 100 条，适合高投入决策

建议：首次分析选标准版，如果结论不够清晰再补充到深度版。"
```

#### 数据采集效率提升建议

```markdown
## 提高评论采集效率的方法

### 方法 1：手机截图 + AI OCR 识别（推荐，比手动复制快 3-5 倍）
- 在手机端浏览评论 → 连续截图 → 发送给我
- 我会自动 OCR 识别并提取评论文本
- 每张截图约可提取 5-10 条评论

### 方法 2：第三方数据工具（免费版即可）
- 蝉妈妈免费版（chanmama.com）— 抖音热门视频评论导出
- 新红数据（xinhong.cc）— 小红书笔记评论分析
- 考拉数据（kalodata.com）— B站视频弹幕/评论
- 以上工具免费版每日有次数限制，但足够采集标准版所需数据量

### 方法 3：直接粘贴
- 手动复制评论粘贴给我（最慢但最简单）
```

### Step 3: 品牌真空检测

**操作**：加载 `engines/brand-vacuum-detector.md` 执行品牌真空检测。

> 如果 `engines/brand-vacuum-detector.md` 尚未创建，使用以下简化流程：

#### 品牌提及率分析

从 Step 1 采集的评论中统计品牌提及情况：

```
品牌提及率 = 提及任何品牌名称的评论数 / 总评论数

判定：
- 品牌提及率 < 5% → 品牌真空，用户无品牌偏好，极佳的品牌建设机会
- 品牌提及率 5-15% → 品牌弱势，有少数品牌但心智不牢固
- 品牌提及率 > 15% → 品牌密集，需要强差异化策略

最低有效样本量：
- 总评论数 < 30 条 → 自动标记置信度为 LOW，在报告中注明"样本量不足，结论仅供参考"
- 总评论数 30-80 条 → 置信度 MEDIUM
- 总评论数 > 80 条 → 置信度 HIGH
```

**退出信号报告**：

| 指标 | 数据 | 判定 |
|------|------|------|
| 转卖原因聚类 | {reasons} | {正常/预警} |
| 全新未拆占比 | {pct}% | {高(>30%)=冲动消费品类⚠️ / 低(<10%)=正常} |
| 二手保值率 | {rate}% | {>60%高保值✅ / 30-60%正常 / <30%溢价虚高⚠️} |
| 转卖量趋势 | {↑/→/↓} | {↑=退出增加⚠️ / →=正常 / ↓=留存好✅} |

**二手保值率计算**：
```
二手保值率 = 闲鱼/メルカリ均价 ÷ 原价(电商平台零售价)

判定：
- > 60%: 高保值 ✅ — 产品价值被市场认可
- 30-60%: 正常 — 合理折旧
- < 30%: 溢价虚高 ⚠️ — 原价可能包含过多品牌溢价或营销泡沫
```

## 失败路径

### 情况 1: 品牌提及率 > 15%（品牌密集）

```
不直接 No-Go，转差异化策略：
↓
分析品牌心智地图中的薄弱环节：
- 有无品牌形象老化？
- 有无价格带空白？
- 有无细分人群未被服务？
↓
找到差异化方向 → 标记为"差异化切入"模式，继续 Phase 4
找不到差异化方向 → 返回 Phase 2，该产品竞争壁垒过高
```

### 情况 2: 用户评论采集不足

```
可能原因：产品太小众 / 用户配合度低
↓
行动：
- 扩大搜索到上级品类的评论
- 使用竞品的评论作为代理数据
- 降低数据量要求，但标注"数据量有限，结论置信度中等"
```

### 情况 3: 退出信号预警（全新未拆占比 > 30%）

```
提示用户：
"⚠️ 该产品在闲鱼/メルカリ上全新未拆封的转卖比例较高（{pct}%），
说明消费者冲动购买后后悔的比例不低。

这意味着：
1. 可能是'种草效果强但实际体验差'的品类
2. 也可能是'礼品属性强'的品类（收到不需要的礼物转卖）

建议在 Phase 5 特别注意：
- 强化产品实际体验，缩小期望差
- 控制退货率预留到 15%+
- 详情页如实展示产品，避免过度美化

是否继续？"
```

---

## 注意事项

1. 用户评论是最有价值的数据源，宁可花时间多采集，不要跳过这一步
2. 差评比好评更重要——差评揭示了真实痛点和改进机会
3. 品牌真空检测的结果直接影响 Phase 4 的品牌策略：真空越大，品牌建设空间越大
4. 退出信号分析是独特的视角，大多数竞品分析都忽略了这一维度
5. 日本市场的品质期待通常高于中国市场，需要额外关注包装、细节和说明书的品质
6. 所有分析结论都需要有数据支撑，避免纯 AI 臆测
